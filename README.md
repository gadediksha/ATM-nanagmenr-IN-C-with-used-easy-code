# ATM Management System in C

This project is a simple ATM Management System written in C language.
It is beginner friendly and menu driven.

## Features
- User Authentication using PIN
- Check Account Balance
- Withdraw Money
- Deposit Money
- Mini Statement (Last 5 Transactions)
- Change PIN Facility
